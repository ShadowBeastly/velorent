RentCore Booking Widget

Local development test — run npm run build in widget/ first.

Test-Anleitung:
1. cd widget && npm install && npm run build
2. dist/embed.js wird erzeugt.
3. Ersetze YOUR_ORG_ID mit einer echten Org-UUID aus Supabase.
4. Starte den Next.js Dev-Server: npm run dev im Root.
5. Öffne diese Datei: open widget/test.html oder serviere lokal.
6. Stelle sicher, dass widget_enabled = true für die Org in Supabase gesetzt ist.
7. CORS: Für lokale Tests bleibe widget_allowed_domains leer (allow-all).
